# xv6 Lottery Scheduler Extension

Dissertation extension implementing lottery scheduling in xv6, built from MIT’s vanilla xv6-public base.

- **Goal:** Replace round-robin with a probabilistic lottery scheduler.
- **Status:** In progress—see commits for lottery implementation.
- **Run:** `make qemu`, then test with `lotterytests` (TBD).
